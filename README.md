### Обо мне

Я занимаюсь системной разработкой, пишу производительный код и раскачиваю навыки в балдеже.  
Мой основной инструмент — Arch Linux (на базе EndeavourOS) с кастомизированным окружением.  
В веб-разработке предпочитаю FastAPI для создания быстрых и надёжных бэкендов на Python.

- Сейчас работаю над: высоконагруженными системами и красивыми интерфейсами  
- Изучаю: системное программирование (C++/C), оптимизацию, паттерны проектирования  
- Спрашивай меня о: C++, Python, FastAPI, архитектуре ПО, Vite  
- Факт: я верю, что код — это искусство, а архитектура — поэзия, и всё это работает на Arch
